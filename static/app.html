API 使用说明

对话补全（Chat Completions）

POST /v1/chat/completions
Authorization: Bearer sk-xxxxxxxx
Content-Type: application/json

{
  "model": "grok-4",
  "messages": [{"role": "user", "content": "Hello!"}],
  "stream": true
}

支持的文本模型

  • grok-3 - Grok 3 标准
  • grok-3-fast - Grok 3 快速
  • grok-4 - Grok 4 标准
  • grok-4-mini - Grok 4 Mini（思维链）
  • grok-4-fast - Grok 4 快速
  • grok-4-heavy - Grok 4 深度
  • grok-4.1 - Grok 4.1 标准
  • grok-4.1-fast - Grok 4.1 快速
  • grok-4.1-expert - Grok 4.1 专家
  • grok-4.1-thinking - Grok 4.1 思维链

图片生成（via Chat API）

POST /v1/chat/completions
Authorization: Bearer sk-xxxxxxxx
Content-Type: application/json

{
  "model": "grok-image-16_9",
  "messages": [{"role": "user", "content": "一只可爱的猫咪"}],
  "stream": true
}

支持的图片模型

  • grok-image - 1:1 默认
  • grok-image-1_1 - 1:1 正方形
  • grok-image-2_3 - 2:3 竖版
  • grok-image-3_2 - 3:2 横版
  • grok-image-16_9 - 16:9 宽屏
  • grok-image-9_16 - 9:16 竖屏

视频生成（via Chat API）

POST /v1/chat/completions
Authorization: Bearer sk-xxxxxxxx
Content-Type: application/json

{
  "model": "grok-video",
  "messages": [{"role": "user", "content": "一只猫咪在草地上奔跑"}],
  "stream": true
}

支持的视频模型

  • grok-video - 16:9 默认
  • grok-video-1_1 - 1:1
  • grok-video-2_3 - 2:3
  • grok-video-3_2 - 3:2
  • grok-video-16_9 - 16:9
  • grok-video-9_16 - 9:16
